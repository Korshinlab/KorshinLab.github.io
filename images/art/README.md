# Artwork images

Put Gregory Korshin's oil painting image files in this folder.

The PI page (`_members/korshin.md`) expects these filenames:

- `korshin_1.jpg`
- `korshin_2.jpg`
- `korshin_3.jpg`

To add more (or fewer) paintings, edit the `gallery.html` block in
`_members/korshin.md` — add/remove `imageN="..."` / `tooltipN="..."` lines.

Recommended: JPG or PNG, roughly 800–1600 px on the long edge.
